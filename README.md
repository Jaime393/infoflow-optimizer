# InfoFlow Optimizer

A new optimizer based on information gradients.

## 🚀 Features
- More stable than Adam
- Better convergence
- Easy to use

## 📦 Install
pip install torch torchvision

## 🧪 Run benchmark
python benchmark.py

## 📊 Goal
Compare InfoFlow vs Adam on MNIST.

## 💡 Idea
Uses normalized information gradient instead of raw gradients.
